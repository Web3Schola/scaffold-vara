This is c/o "Web 3 Schola"
built in 2024 for Varathon.

## Quickstart commands:

1. ``` git clone git@github.com:Web3Schola/scaffold-vara.git ``` 
2. ``` yarn install ```
3. To run the NextJS app, run:
``` yarn start ```
4. To run the local chain, run:
``` gtest crate ```
